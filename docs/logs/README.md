# Logs

Convergence logs capture timestamped execution history, rationale records, and milestone closeout evidence.

## Rules

- Log files live under `YYYY-MM/` month shards.
- File names use `DD-HHMMSS-slug.md`.
- Use logs for meaningful work batches, decisions, and rationale updates.
- Do not create a parallel `decisions/` tree.
