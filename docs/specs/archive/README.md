# Specs Archive

Archive closed or paused strict-lane specs here once they no longer belong in
the active `docs/specs/` tree.

## Working Rule

- active `docs/specs/` should stay focused on live or near-live planning
- archive preserves planning history and traceability
- archived specs are not live execution authority

## Layout

- `NNN-<slug>.md` for archived strict-lane specs
- `batch-cards/NNN-<slug>.md` for archived batch cards from closed or paused
  lanes

## Next Task

Keep the active specs tree small enough that operators can identify the live
strict lanes without wading through closed history.
