# Archived Batch Cards

Use this folder for closed or paused batch cards that no longer belong in the
active `docs/specs/batch-cards/` tree.

## Working Rule

- archive cards after the governing lane is closed or paused cleanly
- do not treat archived cards as the current ready queue
- keep references intact so old logs and specs stay traceable
