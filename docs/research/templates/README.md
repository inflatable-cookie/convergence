# Research Templates

Reusable templates for Convergence's research and implementation-traceability workflow.

## Available Templates

- `specimen-dossier-template.md`
- `value-track-synthesis-template.md`
- `translation-memo-template.md`
- `source-hub-template.md`
- `discovery-intake-template.md`
- `discovery-triage-log-template.md`
- `implementation-decision-record.md`

## Working Rule

Instantiate the dossier and source-hub templates first, then promote repeated findings into a value-track synthesis before writing a translation memo.
Use the implementation decision record when a build decision needs durable traceability back to translation memos, value tracks, dossiers, or prototype findings.

## Next Task

Add additional research templates here only when the current research workflow actually repeats enough to justify them.
